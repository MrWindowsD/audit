# Project Viski — «Оптимизатор сети»

Это основной индекс документации проекта. Здесь описаны архитектура, роадмап и текущий статус разработки.

## 🚀 Быстрый старт (Сборка)

1. **Требования:** JDK 17, Android SDK, NDK (`30.0.14904198`), Go `1.26+`.
2. **Сборка Go-ядра:** В папке `go-core` выполните `.\build.ps1`. Библиотеки `libcore.so` будут разложены по ABI в `app/src/main/jniLibs/`.
3. **Сборка APK:** Выполните `.\gradlew.bat assembleDebug`.

## 🛠 Ключевые достижения текущего билда

*   **Надежная остановка:** Внедрена система отмены контекстов в Go, гарантирующая мгновенное закрытие всех соединений.
*   **Исправление fdsan:** Решена критическая проблема владения файловым дескриптором TUN через `detachFd()`.
*   **Реактивный UI:** Добавлены состояния загрузки и полноценная кнопка управления ядром.
*   **Динамический DNS:** Адрес DNS-сервера теперь передается в ядро через JSON-конфигурацию из Kotlin.
